# Grocery Store Management System

## Project Description
This is a C++ based console application developed as a First Semester project.

## Features
- Add Items
- Update Items
- Delete Items
- Display Inventory
- Calculate Total Cost
- Save & Load Inventory from File

##  Language Used
- C++
- File Handling
